# Animations Folder

Place your Lottie animation JSON files here.

## For the Goddess Animation

1. Create or obtain a Lottie animation (JSON format)
2. Name it `goddess.json` and place it here
3. The LottieGoddess component will automatically load it from `/animations/goddess.json`

## Finding/Creating Lottie Animations

- **LottieFiles**: https://lottiefiles.com (free and paid animations)
- **After Effects + Bodymovin**: Export from After Effects using the Bodymovin plugin
- **Figma + LottieFiles plugin**: Design and export from Figma

## Recommended Animation Properties

- Loop: Yes (for idle/speaking states)
- Size: 400x400px or larger (will be scaled)
- Duration: 2-4 seconds per loop
- Style: Goddess/feminine figure, gold/amber tones preferred
